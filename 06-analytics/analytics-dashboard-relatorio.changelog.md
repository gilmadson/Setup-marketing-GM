# Changelog - Analytics: Dashboard e Relatorio

## [1.0.0] - 2026-05-01
- Looker Studio: conectores nativos (GA4/Google Ads) e de terceiros para Meta (Supermetrics/Porter Metrics)
- Estrutura de 5 páginas: Visão Geral / Meta Ads detalhe / Google Ads detalhe / SEO orgânico / Funil de conversão
- Relatório mensal com 7 seções narrativas: resumo executivo → resultados → o que funcionou → o que não funcionou + por quê → ajustes realizados → plano próximo mês → métricas de vaidade
- Comparações de período: MoM, YoY, vs Meta com código de cores (verde/amarelo/vermelho)
- Princípios de storytelling: contexto antes de número, causa antes de dado, problema com solução
- Frequência de relatório: semanal (primeiros 30 dias), mensal (campanha madura), trimestral (revisão estratégica)
- Template de relatório mensal em Markdown pronto para uso
- Armadilhas: relatório sem insights, métricas de vaidade primeiro, esconder dado ruim, dashboard manual, sem acesso do cliente
