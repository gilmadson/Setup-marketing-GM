# Changelog - SEO: Off-page e Link Building

- Qualidade de backlink: relevância, posição na página, texto âncora, DoFollow vs NoFollow vs Sponsored vs UGC
- 7 técnicas de link building em ordem de eficácia: Digital PR, Guest Post, Broken Link Building, HARO/Connectively, Menções sem link, Parceria, Link Magnets
- Brand signals: buscas pela marca, menções, presença em redes sociais, reviews — como construir
- Toxic links: identificação via Ahrefs, quando usar disavow file (e quando NÃO usar)
- Disavow file: formato correto, processo de submissão, riscos de uso incorreto
- Análise de concorrentes: Site Explorer + Content Gap para encontrar oportunidades de link building
- Template de outreach para guest post
- Armadilhas: compra de links em massa, âncora exata repetida, guest posts em farms, disavow preventivo
