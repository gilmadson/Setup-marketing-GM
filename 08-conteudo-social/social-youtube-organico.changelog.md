# Changelog - Conteudo Social: YouTube Organico

## [1.0.0] - 2026-05-01
- Algoritmo por fase: 0-1k (SEO de pesquisa), 1k-10k (CTR + retenção), 10k+ (volume + consistência)
- SEO de vídeo: pesquisa de keyword via TubeBuddy/VidIQ/Autocomplete, título (60 chars, keyword no início), descrição (keyword nas 2 primeiras linhas, timestamps, links), tags (5-15), capítulos indexados pelo Google
- Thumbnail: 5 princípios (contraste, rosto expressivo, texto 3-5 palavras, clareza, consistência visual), ferramentas (Canva), A/B test via YouTube Studio Beta (10k+ inscritos)
- Estrutura de vídeo para retenção: abertura sem "bem-vindo" (hook direto), pattern interrupts a cada 2-3 min, promessas de conteúdo futuro, CTA no meio do vídeo, End Screen + Cards
- Duração ideal por tipo: tutoriais (8-15 min), entrevistas (30-60 min com capítulos), listas (6-12 min)
- YouTube Shorts: independência do algoritmo principal, estratégia de repurposing + nativo, 3-5 Shorts/semana
- Monetização YPP: requisitos (1.000 inscritos + 4.000h), fontes além do AdSense (membros, brand deals, afiliados, produtos próprios)
- Analytics: CTR (benchmark >4%), AVD (benchmark >40%), Watch Time, Traffic Sources, Audience Retention Graph para diagnóstico
- Template de briefing de roteiro com estrutura completa
- Armadilhas: thumbnail descuidada, introdução longa, keywords sem pesquisa, inconsistência, ignorar Analytics, canal generalista
