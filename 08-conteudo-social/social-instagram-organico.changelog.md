# Changelog - Conteudo Social: Instagram Organico

## [1.0.0] - 2026-05-01
- Algoritmo 2026: 4 fatores (relação, relevância, popularidade inicial, formato priorizado) com sinais de distribuição (saves > compartilhamentos > comentários > watch time)
- Reels: especificações técnicas (9:16, 1080x1920, 15-90s), hook em 3 segundos (4 tipos de hook), estrutura de 4 partes, SEO de Reel via legenda + hashtags
- Carrossel: estrutura de 8-10 cards (capa → conteúdo → CTA), tipos que performam (listas, antes/depois, tutorial), specs 1:1 ou 4:5
- Stories: tipos de interação (enquetes, quiz, caixa de perguntas, link, contagem regressiva), frequência 5-10/dia, Highlights organizados
- Hashtags: mix recomendado (nicho + audiência + conteúdo), máximo 3-8, SEO via legenda como fator mais importante
- Bio otimizada: estrutura de 3 linhas + link, campo "Nome" para SEO (indexado na busca), distinção @username vs Nome
- Horários: janelas por período (manhã/almoço/noite) + instrução para usar Insights da conta específica
- Métricas: saves, alcance, taxa de engajamento (benchmark 1-6%), crescimento, profile visits, link clicks
- Template de briefing de Reel com roteiro estruturado
- Armadilhas: postar sem estratégia, ignorar Reels, hook fraco, inconsistência, Stories como depósito de posts, hashtags banidas
