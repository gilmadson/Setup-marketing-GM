# Changelog - Criativos: Copy para Anúncios

## [1.0.0] - 2026-05-01
- Skill completa — frameworks e técnicas de copy para anúncios em todas as plataformas
- 4 frameworks: PAS (Problem-Agitate-Solution), AIDA, BAB (Before-After-Bridge), 4Ps — com exemplos completos
- Headlines: 6 fórmulas (pergunta, número específico, resultado+prazo, revelação, urgência, personalização por público)
- Limites de caracteres por plataforma: Meta (125/40 chars), Google Search (30/90 chars), TikTok, LinkedIn
- 5 gatilhos emocionais: urgência, escassez, prova social, autoridade, pertencimento — com regras de uso ético
- CTAs por objetivo: awareness, lead, demo B2B, compra direta, app — com exemplos de linguagem de resultado
- Protocolo de teste A/B: o que testar em ordem de impacto, regras de validade estatística
- Template de copy para anúncio com campos para versão A e versão B de teste
